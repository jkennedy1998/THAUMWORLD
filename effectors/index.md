# effectors
> how numbers can change

EFFECTOR is a change in the ROLL.

There is no limit to how many EFFECTORS
can be on a ROLL at once.

EFFECTOR TYPES are listed below.
This is the maths,
and the maths is done in this order.

## EFFECTOR TYPES

SCALE
