# sleep-n-repair
> action types that heal you

A CHARACTER must use the SLEEP or REPAIR ACTION
4 times in every day.

###### 10 EXTENDED ACTIONS occur in a day.

If 10 EXTENDED ACTIONS pass and a CHARACTER has not taken
the required amount of SLEEP or REPAIR,
the following occurs every EXTENDED ACTION.

## HEALTH RECOVERY

HEALTH recovery for SLEEP and REPAIR:

VIGOR MAG ROLL  
+ CON BONUS  
+ any EFFECTORS

## SLEEP based CHARACTERs

If the CHARACTER is above 0 VIGOR :
- take 1 VIGOR DAMAGE

If the CHARACTER is at 0 VIGOR :
- the CHARACTER automatically takes the EXTENDED ACTION: SLEEP

## REPAIR based CHARACTERs

If the CHARACTER is above 0 VIGOR :
- take 1 VIGOR DAMAGE

If the CHARACTER is at 0 VIGOR :
- take 1 DAMAGE of HEALTH
