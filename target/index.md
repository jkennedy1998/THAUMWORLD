# target
> what rules can point at

a TARGET is anything in the world that an ACTION, TAG, or SPELL can refer to.

## what can be a TARGET
- CHARACTER
- BODY SLOT (of a CHARACTER)
- ITEM
- TILE
- REGION TILE
- WORLD TILE

## targeting requirements
to TARGET something, you generally need [ AWARENESS : TARGET ].
> see the AWARENESS rules page and the TAG TABLE

## touching and awareness
if you are touching something then you would have awareness if you have over 0 PRESSURE sense.
things that cant feel touch may not be aware when they are touched. tough they do have other senses.

## TARGETs in spells
SPELLS are made of GLYPHs.
TARGETER GLYPHs output a TARGET.
RE-TARGETER GLYPHs change a TARGET.
SENSOR GLYPHs take a TARGET and output a CONDITION.
a CONDUCTIVE TOOL provides an inherited vector or TARGET for spell casting.

## examples
- ATTACK a TARGET : a CHARACTER
- INSPECT a TARGET : an ITEM
- TARGET a BODY SLOT : ( head ) on a CHARACTER
- TARGET a TILE : the tile is on ( FIRE! )
- TARGET a REGION TILE : the encounter space you are in
- TARGET a WORLD TILE : the region you are traveling to
