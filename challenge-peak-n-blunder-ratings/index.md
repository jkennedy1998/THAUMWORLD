# challenge-peak-n-blunder-ratings
> extraordinary success and failure of result rolls

CHALLENGE RATINGS are a measure of how hard it is to do an ACTION.
they only apply to the RESULT ROLL.

> 0  : no roll, mundane
> 10 : averagely complex
> 20 : pretty hard task
> 50 : moving boulders

a ROLL procs a CHALLENGE RATING if it is above or equal to it.

BLUNDER RATINGS are a mechanic to represent extra-ordinary failure.
a ROLL procs a BLUNDER RATING if it is below or equal to it.

> this value is 1 unless stated otherwise

PEAK RATINGS are a mechanic to represent extra-ordinary success.
a ROLL procs a PEAK RATING if it is above or equal to it.

> this value is 20 unless stated otherwise

PEAK and BLUNDER circumstances are defined for some ACTION types.
these circumstances will always happen, but more things may happen too :3
